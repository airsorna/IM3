Introduction

User interface (UI) and user experience (UX) in the context of web design are the two most important fundamental principles used to make websites interactive for the user. It is a way of ensuring users engage with your website effortlessly and effectively. It is a principle that guides web designers on how to make user-centered websites and web pages. But what are UI and UX? What constitutes good UI and UX? These are the questions I seek to answer in this essay, and I will use Property24 as the basis for my argument and analysis as it relates to UI and UX, interaction, and ethics.

What is UI & UX?

What is User Interface? According to Jair Leite, "User Interface (UI) is a message that communicates to users what they can do with an application and how they can use it." (Leite, 2002). He describes UI using the interaction model, which can be broken into 3 parts:

"The Function Command specifies the way users enter information and control the execution of application function. Interaction results are user to provide feedback to users about function results or error messages. The Browsing Control is an optional component that is used to allow users to control another user interface objects. Pop-up and pull-down menus and the window minimization and maximization controls are examples of browsing control." (Leite, 2002).

User interfaces are all of the functional components the user interacts with. This includes, but is not limited to, all hardware and software used to direct the user’s actions. First, we have "interface signs are the widgets, icons, words, menus, dialog boxes, wizards (assistants), and all interface elements that can convey messages. Interface signs convey designer’s message to users and are also a medium to the interaction activity." (Leite, 2002). Second, "the display medium is the physical medium where all interface signs are conveyed. The interaction activity happens through it. The display medium is usually the computer screen." (Leite, 2002). And lastly, "the activation tool refers to any mechanism that can be used to perform actions using the interface signs in the display medium. Users’ arms and fingers, mouse device, light pens are examples of the activation tool." (Leite, 2002). User 3 engagement is at the core of good UI design, everything needs to be clearly labelled and specified to ensure users are engaging with the website as intended. What is User Experience? UX refers to the fundamental components of a website that provide knowledge about a product or service. UX is the structural building block on which UI is built. For example, UX makes the website useful, and UI makes it presentable. "UX design focuses on the interaction that a human user has with everyday products and services. The goal of UX design is to make using these products and services, both digital or physical, easy, logical, and fun." (Oppliger, 2021). And it "involves the design of the entire process of acquiring and integrating the product, including aspects of branding, design, usability, and function." (Interaction Design Foundation, 2023). UX is built with a product or service in mind. It’s there to boost consumer interest, as it is the core of the website. Functionality is at the heart of UX design, "where the main objective is to create effective websites where the user quickly and efficiently can obtain the desired pieces of information without being delayed by long downloading times or blind alleys when navigating on the site" (Thorlacius, 2007:64). UI is used to complement UX. Think of UX as the foundation on which UI is built upon.

Property24

Property24 is a South African online property auction company that provides services across the country for homebuyers and real estate agents. They are the leading property website in South Africa, providing top real estate companies with brand exposure and marketing. With an estimated 6 million visitors a month, their website connects real estate companies with homebuyers and renters searching for property online.

Figure 1.1. Property24

I have chosen Property24 because I think they have kept this simple for the sake of functionality. And by functionality, I mean "the user-friendly aspects of interfaces and Human Computer Interaction (HCI), where the main objective is to create effective websites where the user quickly and efficiently can obtain the desired pieces of information without being delayed by long downloading times or blind alleys when navigating on the site." (Thorlacius, 2007:64). This is all brought about by good UX design. The site uses the basic UX design principles, which are:

Von Restorff Effect: make important information visually distinctive. We see this being done with the Filters and More Filters Button, Search Panel, and Order By. This is done to help the user remember the key actions and guide them to where you want them to go.

According to Hick's Law, time spent making decisions increases with complexity. They used this with the filter’s dropdown menu and search bar. This is done to help users navigate through the website and save them time from scrolling through a hundred listings when they need something specific.

Serial Position Effect: key information must be placed at the beginning and end and less important in the middle. At the top, they’ve placed the search panel, the name of the website, filters, and order by. At the bottom are their contact and bio information, along with their social media accounts. In the middle, they have house listings, etc. 5 This is how information is structured in order for users to fully engage with your website.

Call to action (CTA): this involves all the symbols used to call upon the user to perform a certain action. It is a visual communication tool needed to ensure users know how to engage with your website because "visual symbols have become an integral part of our daily lives. Therefore, it is increasingly relevant to understand their communicative effects. The orientation towards visual communication is not only prevalent in marketing but in all forms of professional communication, including the Web site as a medium." (Thorlacius, 2007:66). As a website that serves many people, it becomes important to have these symbols in place in order to save time and keep the visitors cognitive load light. The search, bed, and car symbols are all semiotic devices that communicate a message to the reader, thus increasing user interaction.

The UI design for this website is also very good. First color and font "If we choose a vibrant, warm red colour for the menu, we communicate something different than if we had chosen a calm, cool blue colour. If we have a specially designed typography made for our headlines we leave a more personal impression on our Website than if we choose the most common typography such as, for example, Verdana." (Thorlacius, 2007:63). With the Property24 logo being blue, we see that’s what influenced the color choice for the header, price, and font. The background is white, so the user's attention is centered on the housing and nothing else. The principles of UI are evident here. All information has been centered, everything is clearly labeled, and cognitive load is reduced as users perform the same action over and over again. Overall, in my opinion, Property24 has done an excellent job with their website. It is user-centered and simple, and thus it is a good example of good UI and UX design.

Conclusion

In conclusion, the principles of good UI and UX are integral components needed to make websites interactive. It is without a doubt that user interaction is influenced by how a website is structured and presented, so giving high consideration to how you want your users to engage with your website is the first step all web designers need to take because this will influence how users experience your site and thus affect the product or service you are selling. Property24 has done this to good effect and has 6 kept things simple and easy to understand. It is the job of web designers to aim for this when making a website, keeping the user at the center of their design.
